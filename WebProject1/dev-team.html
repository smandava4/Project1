Project Name:

Adventure Game - Hunt the Dragons

Description:

This is an HTML and CSS project that immerses players in a thrilling first-person view adventure. In this captivating game, player takes on the role of a skilled sniper armed with a powerful sniper rifle to hunt and eliminate menacing dragons. The game features an adventurous theme, providing an adrenaline-pumping experience. With just HTML and CSS, this project offers an engaging gaming experience, showcasing the creative potential of these web technologies.
